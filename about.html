굿노트 스터디 플래너, 이렇게 활용하세요

이 플래너는 아이패드 굿노트·노터빌리티 같은 필기 앱에서 바로 열어 쓸 수 있도록 만든 하이퍼링크 PDF입니다. 표지와 각 페이지 상단의 탭을 누르면 원하는 섹션으로 바로 이동합니다. 연간 목표부터 월간·주간·일간 계획까지 한 파일 안에서 관리할 수 있습니다.

이런 분들께 추천합니다

  • 아이패드로 필기하는 수험생 — 과목별 진도와 시간표를 손글씨로 정리하고 싶은 분
  • 여러 과제와 시험이 겹치는 대학생 — 월간·주간·일간 단위를 넘나들며 계획하고 싶은 분
  • 습관 관리를 함께 하고 싶은 분 — 뽀모도로, 기분, 습관 트래커로 하루하루를 기록하고 싶은 분

단계별 활용법

  1. 표지에서 이동하기 — 표지 오른쪽 탭을 눌러 나의 목표, 캘린더, 시간표 등 원하는 섹션으로 바로 이동합니다.
  2. 연간 목표·캘린더 채우기 — 나의 목표 페이지에 올해 목표를 적고, 캘린더 페이지에서 각 월을 누르면 월간 달력으로 이동합니다.
  3. 월간 → 위클리 → 데일리 순으로 계획 — 월간 달력에서 이달의 할 일을 정리하고, 위클리 플랜에서 주간 목표를, 데일리 로그에서 오늘의 할 일과 뽀모도로 세션을 기록합니다.
  4. 습관 기록 확인 — 월간 달력 하단 링크를 눌러 습관 기록 페이지로 이동해 한 달의 루틴을 체크합니다.

필기 앱 사용 팁

굿노트나 노터빌리티에서 PDF를 가져오기(Import)한 뒤, 표지나 각 페이지의 탭을 터치하면 해당 섹션으로 바로 이동합니다. 매년 새 플래너가 필요하면 이 사이트에서 다시 다운로드해 새로 가져오기 하시면 됩니다.
