# 스터디 플래너 PDF 무료배포 + 애드센스 랜딩페이지 설계

## 배경 및 목적

스터디 플래너 PDF를 무료로 배포하는 랜딩페이지를 만들고, 구글 애드센스 광고를 붙여 수익을 창출한다. 애드센스는 아직 신청/승인 전이므로, 승인 심사를 통과할 수 있도록 콘텐츠를 갖춘 사이트를 먼저 구축하는 것이 이번 프로젝트의 핵심 목표다.

## 범위

- 스터디 플래너 PDF 1종(단일 디자인) 제작
- PDF를 무료로 다운로드할 수 있는 정적 랜딩페이지 제작
- GitHub Pages를 통한 무료 배포 (별도 도메인 구매 없음)
- 애드센스 심사를 통과하기 위한 최소 콘텐츠(소개, 개인정보처리방침 페이지) 구비
- 애드센스 광고 스크립트를 나중에 꽂아 넣을 수 있는 광고 슬롯 마련

### 범위 밖 (Out of scope)

- 애드센스 계정 신청/심사 절차 자체 (사용자가 직접 진행)
- 여러 종류의 플래너 템플릿 제작
- 이메일 수집/뉴스레터 등 리드 제너레이션 기능
- 커스텀 도메인 구매 및 연결
- 방문자 트래킹/분석 도구 연동 (GA 등)

## 기술 스택 및 접근 방식

순수 정적 HTML/CSS/JS 사이트. 빌드 도구나 서버 없이 GitHub Pages에 그대로 배포 가능한 구조를 사용한다. 스터디 플래너 PDF도 HTML/CSS로 디자인한 뒤 브라우저 인쇄(PDF로 저장) 기능으로 생성하여, 랜딩페이지와 디자인 톤을 통일한다.

## 페이지별 콘텐츠

### index.html (메인 랜딩페이지)
- 플래너 미리보기 이미지
- 특징 소개: 주간 계획표, 과목별 체크리스트, 목표란 등
- 다운로드 버튼 (클릭 시 즉시 `downloads/study-planner.pdf` 다운로드, 이메일 입력 없음)
- 상단/하단 광고 슬롯 자리 (`<div class="ad-slot">`, 승인 전에는 빈 상태 또는 플레이스홀더)

### about.html (소개/활용법)
- 플래너 사용법 안내
- 추천 대상(수험생/대학생 등)
- 활용 팁
- 애드센스 심사 시 "실질적 콘텐츠"로 인정받기 위한 텍스트 분량 확보 목적

### privacy.html (개인정보처리방침)
- 쿠키 및 광고 관련 개인정보 처리 안내 문구 포함 (애드센스 심사 필수 항목)

## 플래너 PDF 디자인 (1종)

A4 세로, 인쇄 친화적 흑백/투톤 디자인:
- 상단: 주(週) 표시 + 목표 한 줄
- 요일별 시간표 그리드 (월~일)
- 과목별 체크리스트
- 하단: 오늘의 한마디/메모란

`planner/planner.html`로 작성하고, 브라우저 인쇄(PDF로 저장) 기능을 통해 `downloads/study-planner.pdf`를 생성한다.

## 애드센스 관련 처리

- 애드센스가 아직 미승인 상태이므로, 이번 구현에서는 광고 스크립트를 실제로 삽입하지 않는다.
- 광고가 들어갈 위치에는 빈 `<div class="ad-slot">` 컨테이너만 마련해 두어, 승인 후 발급받은 스크립트로 손쉽게 교체할 수 있도록 한다.
- `ads.txt` 파일은 애드센스 승인 후 발급되는 퍼블리셔 ID를 반영해야 하므로, 이번 구현에서는 생성하지 않거나 안내 주석만 남긴다.
- 애드센스 신청 자체는 GitHub Pages 배포로 실제 URL이 생긴 뒤 사용자가 직접 진행한다.

## 배포 절차

1. 새 GitHub 저장소 생성 (저장소 이름은 진행 시 확인)
2. 로컬 코드를 저장소에 push
3. 저장소 Settings → Pages에서 배포 활성화
4. `https://<username>.github.io/study-planner-site/`로 접속 확인

## 테스트/검증 방법

- 로컬에서 정적 파일을 브라우저로 열어 다운로드 버튼, 페이지 링크(about/privacy) 동작 확인
- PDF가 정상적으로 열리고 인쇄 레이아웃이 A4에 맞는지 확인
- GitHub Pages 배포 후 실제 URL에서 동일하게 동작하는지 확인
- 모바일/데스크톱 반응형 레이아웃 확인
